About This Project

This website was created as a project for a computer science class.

The topic chosen for the project is a quick overview of the most basic steps of adult CPR.

The goal of the site is to present simple CPR concepts in a clear and easy-to-read format.

This project was created quickly and somewhat on the fly as part of the learning process while building the website.

The information included covers only the very basic aspects of CPR and should not be considered complete medical training.

If you are interested in learning CPR properly, consider taking a certified CPR course from a qualified training organization.

About the Creator

My name is Hasim Tarik and I am a student at the University of Hawaiʻi at Hilo.

I created this website as part of a class project to learn more about HTML, CSS, and web design principles. It was a fun and educational experience to build this site and share information about CPR.

In my free time, I enjoy learning about various health subjects without being told to.
